# 教程 01 - 理解 Agent

> 什么是 AI Agent？它和 ChatGPT 有什么区别？

---

## 什么是 Agent？

**一句话**: Agent 是能自主完成任务的 AI 程序。

**类比**:
- **ChatGPT** = 你问一句它答一句（被动）
- **Agent** = 你给它一个目标，它自己规划、执行、调整（主动）

就像你告诉助理 "帮我整理会议纪要"，助理会自己决定：
1. 先听录音转成文字
2. 提取关键信息
3. 整理成结构化文档
4. 发给参会人

这就是 Agent 的工作方式。

## Agent 的三大能力

```
         ┌─────────────┐
         │   LLM 大脑   │  ← 核心驱动力
         └──────┬──────┘
                │
    ┌───────────┼───────────┐
    │           │           │
    ▼           ▼           ▼
┌────────┐ ┌────────┐ ┌────────┐
│ Planning│ │ Memory │ │ Tool   │
│ 规划能力│ │ 记忆能力│ │ 工具能力│
└────────┘ └────────┘ └────────┘
```

1. **Planning（规划）**: 把大任务分解成小步骤
2. **Memory（记忆）**: 记住之前的对话和操作
3. **Tool Use（工具）**: 调用外部 API、数据库等

## 单Agent vs 多Agent

### 单Agent

一个 Agent 做所有事情。

**问题**: 任务太复杂时容易"注意力漂移"——就像让一个人同时做翻译、写总结、分析数据，质量会下降。

### 多Agent（本项目采用）

多个专业化 Agent 分工协作。

**优势**:
- 每个 Agent 专注一件事，质量更高
- 独立任务可以同时执行，速度更快
- 一个 Agent 出问题不影响其他的

## 核心概念: ReAct 模式

Agent 的思考方式叫 **ReAct**（Reason + Act）：

```
循环:
  1. 思考 (Thought): "我应该做什么？"
  2. 行动 (Action): 调用工具执行
  3. 观察 (Observation): 看看结果如何
  4. 判断: 任务完成了吗？
     - 是 → 结束
     - 否 → 回到第1步
```
